Funções como valores

Lambda

Trecho de código executável que pode ser tratado como um valor.

Sem Lambda (Java 7)

public class ListOp{ public boolean dropLower( List<Integer> values, int lowerLimit ){ List<Integer> result = new ArrayList<>(); for( int value: values ){ if( value >= lowerLimit ) result.add(value); } return result; } public boolean dropHigher( List<Integer> values, int upperLimit ){ List<Integer> result = new ArrayList<>(); for( int value: values ){ if( value <= lowerLimit ) result.add(value); } return result; } }

Refatoração com Lambda (Java 8)

interface Predicate<T>{ boolean apply(T value); } public class ListOpLambda{ public boolean dropLower( List<Integer> values, int lowerLimit ){ return dropWithPredicate( values, (i) -> (i >= lowerLimit) ); } public boolean dropHigher( List<Integer> values, int upperLimit ){ return dropWithPredicate( values, (i) -> (i <= upperLimit) ); } private boolean dropWithPredicate( List<Integer> values, Predicate<Integer> predicate ){ List<Integer> result = new ArrayList<>(); for( int value: values ){ if( !predicate.apply(value) ) result.add(value); } return result; } }

Mudancça da API para utilizar Lambdas

public class ListOpLambdaApi{ public boolean dropLower( List<Integer> values, int lowerLimit ){ return values.filter( (i) -> (i >= lowerLimit) ); } public boolean dropHigher( List<Integer> values, int upperLimit ){ return values.filter( (i) -> (i <= lowerLimit) ); } }
